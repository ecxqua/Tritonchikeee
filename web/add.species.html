Создание новой карты

Выберите шаблон описания карточки (ИК-1, ИК-2, КВ-1, КВ-2). Поля можно заполнять частично и править повторно; черновик сохраняется локально в браузере до подключения API и БД.

Пустые поля допускаются: данные можно дозаполнить позже. Для ИК-2 ID родителей можно связать с базой — после ввода ID доступна ссылка на карту в разделе «Проекты тритонов».
