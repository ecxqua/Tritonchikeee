# Краткая структура независимого пайплайна

`config/` - папка конфигураций для запуска анализа. Используйте её для "игры" с параметрами.

`data/` - датасет, ввод и вывод анализа.

`docs/` - документация.

`pipeline/` - основной пайплайн.

`models/` - актуальные веса моделей.

`train/` (не проверено) - обучение моделей.

`bot/` - сервис тг-бота.
